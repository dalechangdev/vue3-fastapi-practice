# Vue3 + FastAPI Practice

This repo houses two projects: a Vue 3 project that uses the Composition API, and a FastAPI project.

It is designed to help me practice for an upcoming full-stack engineering coding challenge.

In this repo, I will practice fundamentals such as:

- TypeScript: interfaces, generics, utility types (Partial, Pick, Omit)
- Vue 3 Composition API:`ref`,`reactive`,`computed`,`watch`
- Component props + emits with typed contracts
- `defineProps<T>`and`defineEmits`patterns
